Proyecto de Realidad Mixta para la Enseñanza del Modelo Atómico
Descripción general

Este proyecto consiste en el desarrollo de un prototipo de aplicación de realidad mixta
orientado a la enseñanza de conceptos científicos, específicamente el modelo atómico,
con un enfoque en accesibilidad para personas con discapacidad visual.

El sistema fue desarrollado utilizando Unity como motor principal, el SDK de XREAL para la
integración con dispositivos de realidad aumentada, y Android Studio para la compilación
y ejecución en un dispositivo móvil Samsung Galaxy S24 Ultra, el cual actúa como unidad
de procesamiento para los lentes XREAL Air 2 Ultra.

Objetivo del proyecto

El objetivo principal es explorar el uso de tecnologías de realidad mixta como herramienta
educativa inclusiva, permitiendo representar conceptos abstractos mediante modelos 
tridimensionales interactivos que puedan mejorar la comprensión, especialmente en contextos 
donde los métodos tradicionales presentan limitaciones.

Tecnologías utilizadas
Unity como motor de desarrollo
XREAL SDK para integración de funcionalidades de realidad aumentada
Android Studio para compilación y despliegue
Lentes XREAL Air 2 Ultra
Dispositivo Samsung Galaxy S24 Ultra
Modelado 3D para la representación del átomo
